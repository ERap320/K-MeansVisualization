# K-MeansVisualization
A graphical representation of the [K-Means clustering algorithm](https://it.wikipedia.org/wiki/K-means) leveraging OpenMP for parallelization

## Building
### Debian based
```
sudo apt-get install libsfml-dev
make
```
### Arch based

```
sudo pacman -S sfml
make
```

## Running
```
./kmeans-gui -h
```
